CS325: Artificial Intelligence
=====

This is an advanced course in computer science, studying topics in Artificial Intelligence especially on Machine Learning and Natural Language Processing. This course will teach machine learning algorithms such as Hidden Markov, Maximum Entropy, Perceptron, and K-means, as well as their applications in natural language processing such as Segmentation, Part-of-speech Tagging, and Document Categorization. Advanced topics such as Viterbi and Expectation–Maximization algorithms will be also discussed. Prerequisites: CS171.
